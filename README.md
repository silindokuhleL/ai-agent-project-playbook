# ai-agent-project-playbook

Shared engineering playbook for AI-powered portfolio projects.

This repository defines the rules agents should follow when building Laravel REST APIs, Next.js frontends, AI-assisted workflows, project documentation, tests, and deployment-ready proof projects.

## Purpose

Use this repo as the shared source of truth for:

- Agent behavior.
- Branching and commit rules.
- Laravel backend standards.
- Next.js frontend standards.
- REST API contracts.
- AI feature standards.
- Testing expectations.
- Project completion checklists.

## First Projects Using This Playbook

- `launchbill-frontend`
